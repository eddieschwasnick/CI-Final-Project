# CI-Final-Project

**Do Information Inputs Causally Affect LLM-Generated Investment Portfolios?**
A randomized experiment on information conditioning across three frontier LLMs,
paired with a real April-2025 → April-2026 backtest.

**Author:** Edward Schwasnick
**Course:** STAT 6990 — Causal Inference, Spring 2026 (Prof. Zhang)

---

## Overview

This project runs a 4-arm randomized experiment on three comparable-tier
frontier LLMs:

- Anthropic `claude-sonnet-4-6`
- OpenAI `gpt-4o`
- Google `gemini-2.5-pro`

In each trial, the model is asked to allocate capital across 20 U.S.
large-cap stocks. The only thing that varies across conditions is the
information block injected into the prompt:

| Condition | Prompt contains |
|---|---|
| `control` | ticker list only |
| `fundamental` | + fundamentals table (market cap, P/E, growth, leverage) |
| `technical` | + technicals table (returns, vol, beta, MA signals) |
| `combined` | + both tables |

Each (model, condition) cell is run 30 times → **360 total API calls**.
Portfolio-level outcomes (HHI, sector HHI, weighted beta, weighted volatility,
breadth) are estimated with Welch's *t*-tests, OLS, and 5,000-iteration
permutation placebos. Because each prompt embeds a *historical* April-2025
data snapshot, every portfolio can also be evaluated against actual prices
through April 2026 — a real out-of-sample backtest, not hindsight.

### Headline findings

1. **Information framing is causal.** Each model's individual ATEs on at
   least one outcome are large, tight, and significant at p < 0.001.
2. **Cross-model heterogeneity is the surprise.** Each vendor responds to
   a *different* prompt:
   - Claude → fundamental (HHI rises 64% vs control)
   - Gemini → technical (HHI rises 55% vs control)
   - GPT-4o → neither (HHI moves < 0.003 across all four arms)
3. **Realized returns unify the story.** Despite the allocation-level
   heterogeneity, the realized April-2025 → April-2026 return ranks the
   same way for every vendor: Fundamental (+5.3 pts vs control) > Control
   > Combined > Technical (-12.4 pts vs control).

[`reports/report.tex`](reports/report.tex).

## Reproducing the results

### Environment

```bash
python -m venv venv && source venv/bin/activate
pip install pandas numpy yfinance matplotlib seaborn scipy statsmodels \
            python-pptx litellm python-dotenv pyyaml
```

Put all three provider keys in a `.env` file at repo root:

```
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
GEMINI_API_KEY=...
```

### Full pipeline

```bash
# 1. Pull the historical data snapshot. --as-of last-year is shorthand for
#    today minus 365 days; produces master_dataset.csv + forward_prices.csv.
python data/pull_financial_data.py --as-of last-year

# 2. Regenerate the 4 prompt files from the new master_dataset.csv so the
#    prompts and data stay in sync. The data block is labeled "current" so
#    the as-of date does not leak to the LLM.
python build_prompts.py

# 3. Run the 4-arm experiment for all 3 models (~30–45 min, ~$3 in API
#    credits across the three providers; each call uses max_tokens=2048
#    and Gemini gets reasoning_effort="low" so its thinking budget can't
#    starve the JSON output).
python run_multi_ai_experiment.py --runs 30 --rerun-claude

# 4. Causal analysis on the Claude slice (the legacy pipeline reads
#    outputs/experiment_results.csv, which step 3 also writes).
python analysis/compute_metrics.py
python analysis/statistical_analysis.py
python analysis/placebo_tests.py
python analysis/visualizations.py

# 5. Cross-model comparison (HHI scatter, sector mix per model × condition,
#    treatment-effect forest plot).
python analysis/multi_ai_compare.py

# 6. Realized 12-month return backtest. MUST be run after step 5 — both
#    write to fig10_multi_ai_returns.png, and step 6's output (using
#    forward_prices.csv) is the canonical one.
python analysis/realized_backtest.py

# 7. Build deck and PDF.
python reports/build_presentation.py
cd reports && pdflatex report.tex
```

Steps 4–7 are deterministic and read from cached API responses, so the
full analysis pipeline reruns offline in well under a minute.

### Convenience flags

```bash
# Re-run only one provider (e.g. after an API outage). Existing rows for
# the other models are preserved automatically.
python run_multi_ai_experiment.py --runs 30 --only gemini

# Smaller sample for testing
python run_multi_ai_experiment.py --runs 5 --only claude --rerun-claude

# Pull a different historical snapshot
python data/pull_financial_data.py --as-of 2024-04-27
python build_prompts.py
```

## Key results

### Per-model treatment effects (Claude, n = 30 per arm)

| Outcome | Treatment | $\hat\tau$ | 95% CI | p |
|---|---|---|---|---|
| HHI | Fundamental | +0.036 | [0.034, 0.039] | <0.001 |
| HHI | Technical | +0.012 | [0.008, 0.016] | <0.001 |
| Sector HHI | Fundamental | +0.096 | [0.088, 0.104] | <0.001 |
| Portfolio β | Fundamental | +0.231 | [0.224, 0.238] | <0.001 |
| Portfolio β | Technical | −0.141 | [−0.151, −0.132] | <0.001 |
| Portfolio vol (%) | Fundamental | +3.85 | [3.70, 4.00] | <0.001 |
| Portfolio vol (%) | Technical | −2.48 | [−2.65, −2.30] | <0.001 |

### Mean HHI by (model, condition)

| Model | Control | Fundamental | Technical | Combined |
|---|---|---|---|---|
| Claude Sonnet 4.6 | 0.057 | **0.094** | 0.069 | 0.065 |
| OpenAI GPT-4o | 0.057 | **0.058** | 0.056 | 0.055 |
| Gemini 2.5 Pro | 0.065 | 0.065 | **0.101** | 0.070 |

### Realized 12-month return (April 2025 → April 2026)

| Model | Control | Fundamental | Technical | Combined |
|---|---|---|---|---|
| Claude Sonnet 4.6 | 39.3% | **49.7%** | 27.9% | 38.8% |
| OpenAI GPT-4o | 40.0% | **41.4%** | 33.4% | 38.3% |
| Gemini 2.5 Pro | 40.6% | **44.5%** | 22.6% | 32.9% |
| **Pooled** | 40.0% | **45.3%** | 27.5% | 36.7% |

Permutation placebos, within-control placebos, and trimmed-sample
sensitivity checks all corroborate the main findings.
